
Command
    gpt github review : Review GitHub PR with Open AI, and post response as a comment.
        Args:     repository (str): The repo of the PR.     pull_request (str): The PR number.
        access_token (str): The GitHub access token.  Returns:     Dict[str, str]: The response.
        WARNING: This command is in preview. It may be changed/removed in a future release.

Arguments
    --access-token     : The GitHub access token. Set or use GITHUB_TOKEN environment variable.
    --pull-request -pr : The PR number. Set or use PATCH_PR environment variable.
    --repository -r    : The repo of the PR. Set or use PATCH_REPO environment variable.

Global Arguments
    --debug            : Increase logging verbosity to show all debug logs.
    --help -h          : Show this help message and exit.
    --only-show-errors : Only show errors, suppressing warnings.
    --output -o        : Output format.  Allowed values: json, jsonc, none, table, tsv, yaml, yamlc.
                         Default: json.
    --query            : JMESPath query string. See http://jmespath.org/ for more information and
                         examples.
    --verbose          : Increase logging verbosity. Use --debug for full debug logs.
