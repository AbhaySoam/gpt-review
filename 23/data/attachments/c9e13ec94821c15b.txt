
Command
    gpt git commit : Run git commit with a commit message generated by GPT.
        Args:     gpt4 (bool, optional): Whether to use gpt-4. Defaults to False.     large (bool,
        optional): Whether to use gpt-4-32k. Defaults to False.     push (bool, optional): Whether
        to push the commit to the remote. Defaults to False.  Returns:     response (Dict[str,
        str]): The response from git commit.
        WARNING: Command group 'git' is in preview. It may be changed/removed in a future release.

Arguments
    --gpt4             : Use gpt-4 for generating commit messages instead of gpt-35-turbo.
    --large            : Use gpt-4-32k model for generating commit messages.
    --push             : Push the commit to the remote.

Global Arguments
    --debug            : Increase logging verbosity to show all debug logs.
    --help -h          : Show this help message and exit.
    --only-show-errors : Only show errors, suppressing warnings.
    --output -o        : Output format.  Allowed values: json, jsonc, none, table, tsv, yaml, yamlc.
                         Default: json.
    --query            : JMESPath query string. See http://jmespath.org/ for more information and
                         examples.
    --verbose          : Increase logging verbosity. Use --debug for full debug logs.
